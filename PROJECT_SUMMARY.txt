TROUBLESHOOTING
===============

Q: How do I use my own dataset?
A: Create this directory structure:
   data/
   ├── class1_name/
   │   ├── image1.jpg
   │   └── image2.jpg
   └── class2_name/
       ├── image3.jpg
       └── image4.jpg
   
   Then:
   loader = SymptomDataLoader()
   X, y, classes = loader.load_images_from_directory('data')

Q: Training is too slow, what can I do?
A: 1. Use GPU (check: python -c "import tensorflow as tf; print(tf.config.list_physical_devices('GPU'))")
   2. Reduce image size: (128, 128) instead of (224, 224)
   3. Increase batch size: 64 instead of 32
   4. Use MobileNetV2 instead of ResNet50

Q: Accuracy is low, what should I improve?
A: 1. Add more training data (100+/class minimum)
   2. Increase epochs: 100+ instead of 50
   3. Improve data quality (clear, properly labeled images)
   4. Try different architectures: ResNet50
   5. Adjust learning rate: 0.0001 instead of 0.001

Q: How do I save the model?
A: model.save('models/my_model.h5')
   Then load with: tf.keras.models.load_model('models/my_model.h5')

Q: Can I deploy this model?
A: Yes! Use inference.py:
   from inference import SymptomAnalyzer
   analyzer = SymptomAnalyzer('models/best_model.h5')
